**제작자: asowjdan**

## 한국어

Java/Spring, Python, Docker, 데이터베이스, 알림 센터, 테마 색상을 하나의 사이드바에서 통합 관리하는 VS Code 확장입니다.

### 주요 기능

- **자바/스프링** — 실행 가능한 코드, Spring 컨트롤러, Java Projects, Maven/Gradle 빌드 도구를 사이드바에서 바로 실행·중지·디버그
- **파이썬** — Python 스크립트 실행 및 디버그 지원
- **도커** — Docker Compose 서비스 실행·중지
- **데이터베이스** — DB 연결 설정 관리, 테이블 데이터 조회, ERD 뷰, Redis 키 조회
- **알림 센터** — VS Code 네이티브 알림을 좌측 패널에서 목록으로 관리, 번역 지원
- **커스텀 디자인** — 색감 프리셋 적용 및 제거

### 사용 방법

1. 확장을 설치하고 VS Code를 재시작합니다.
2. 좌측 액티비티 바에서 원하는 아이콘을 클릭합니다.
3. 각 패널에서 실행, 중지, 설정을 조작합니다.

### 주의 사항

- 배경 이미지 기능은 별도의 **Custom Workbench Background Mod** 확장과 함께 사용합니다.
- 색상 설정은 VS Code 전역 설정(`settings.json`)에 저장됩니다.

## English

A VS Code extension that centralizes Java/Spring, Python, Docker, database, notification, and theme color management in a single sidebar.

### Features

- **Java/Spring** — Run, stop, and debug runnable code, Spring controllers, Java projects, and Maven/Gradle tasks directly from the sidebar
- **Python** — Run and debug Python scripts
- **Docker** — Start and stop Docker Compose services
- **Database** — Manage DB connections, browse table data, view ERDs, and inspect Redis keys
- **Notification Center** — Collect and manage VS Code native notifications in the left panel with translation support
- **Custom Design** — Apply and remove color theme presets

### How to Use

1. Install the extension and restart VS Code.
2. Click any icon in the left activity bar.
3. Use the panels to run, stop, and configure each tool.

### Notes

- For background image support, use this extension together with **Custom Workbench Background Mod**.
- Color settings are saved to your VS Code global `settings.json`.
